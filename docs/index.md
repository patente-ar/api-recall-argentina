# API de recall vehicular en Argentina

Documentacion publica para integrar API recall vehicular con patente.ar.

- Endpoint: `https://api.patente.ar/v1/consultas`
- Producto: `recall`
- Entrada: `vin`
- Ejemplo: `8AJXXXXXXXX123456`
